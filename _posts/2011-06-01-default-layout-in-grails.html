Default layout in Grails is evil. Documentation doesn't state any warnings about using default layout, by setting grails.sitemesh.default.layout in Config.groovy. But it is really unpredictable what will happen with plugin's templates if default layout will be specified. In my case I've got an unending web-page loaded inside it-self, instead of comments list, when used commentable plugin.
